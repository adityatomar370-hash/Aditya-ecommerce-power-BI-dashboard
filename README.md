# 📊 Aditya Ecommerce Sales Dashboard

## 🚀 Project Overview

This project is an interactive Power BI Ecommerce Dashboard created to analyze sales performance, profit trends, customer behavior, payment modes, and category-wise business insights.

The dashboard helps businesses track KPIs and make data-driven decisions using visual analytics.

---

## 🛠️ Tools & Technologies Used

* Power BI
* DAX
* Data Visualization
* Excel / CSV Dataset

---

## 📌 Key Insights

* Quarterly Sales & Profit Analysis
* Customer-wise Sales Amount
* State-wise Revenue Distribution
* Category & Sub-Category Performance
* Payment Mode Analysis
* Monthly Profit Trends
* Average Order Value (AOV) Tracking

---

## 📈 Dashboard Features

* Interactive Filters & Slicers
* Quarterly Analysis (Qtr 1 - Qtr 4)
* KPI Cards
* Dynamic Charts & Graphs
* Business Performance Insights
